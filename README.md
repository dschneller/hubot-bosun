# hubot-bosun

Allows hubot to interact with Bosun.

See [`src/bosun.coffee`](src/bosun.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-bosun --save`

Then add **hubot-bosun** to your `external-scripts.json`:

```json
[
  "hubot-bosun"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
